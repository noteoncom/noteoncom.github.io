A Florist

Help to take care of plants.

  • Record watering, fertilization, de-worming, pruning or changing pots
  • Take one or more photos, record plant growth, or share it through other apps
  • Add notes to the record to make the flowers maintenance clear
  • Add schedules and set reminders for each required maintenance

If you have a lot of plants, try creating a catalog:

  • Add plants to the catalog
  • Set up a conservation plan for each catalog of plants and set reminders
  • Batch operations on plants in a catalog

In addition, the app has a built-in photo album that allows you to view all photos of all or one plant quickly.

Some features are included in premium version.
